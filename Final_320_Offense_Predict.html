Intro: Creating an NFL offensive playcall predictor¶

Running an NFL defense can be a stressful and time consuming job. Even after all the preperation throughout the week, a defense could go out and get curb stomped for seemingly no reason. This is due to the fact that even if a team prepares all week, and watches all the film, they still don't know what play is going to be called on down by down basis. I want to attempt to solve that by creating a playcall predictor that takes is able to correctly predict if a pass or run is being called.

What is going on here: Here I am doing two things. First I am making both a training set and testing set to train and test the play call predictor. The training set takes the whole dataframe and narrows it down to plays involving 3 key factors:

  1. The plays happened outside of the 2018 season
  2. The play happened in between downs 1-5
  3. The plays were either pass or run, we're not going to be factoring in things like punts, kneels, etc.

The test set consists of the same type of plays except all the plays in the test set are IN the 2018 season. So we're training the predictor with all pass/run plays outside of 2018 and testing it to see if it can predict the 2018 play calls.

Secondly, I graphed the play call breakdown between the test set and training set. This allows us to see how pass heavy vs run heavy the ravens were as a whole during the 2009-2017 time period as well as the 2018 season. This could be beneficial as it could show that while the Ravens were slightly pass heavy, they ran the ball a lot leading to a predictor that favors the run a bit more than if we trained it with another team.

Using the graph as a baseline it can be said that a predictor that predicts if a pass play was called would be accurate 54.23% of the time if trained on the training data.

In [14]:
res = pd.DataFrame(columns =['accuracy'])

for team in df.posteam.unique():
    
    testing_df = df[(df.game_date.str.contains('2018')) & (df.posteam == team) & (df.down.isin(range(1,5))) & ((df.play_type == 'run') | (df.play_type == 'pass'))]
    
    test_feat = testing_df[['yardline_100','quarter_seconds_remaining','half_seconds_remaining',
                                   'game_seconds_remaining','qtr','down','goal_to_go','ydstogo',
                                   'score_differential','shotgun','no_huddle']]
    test_label = testing_df['play_type']
  
    prediction = gbr.predict(test_feat)
    accuracy = accuracy_score(test_label, prediction)

    res.loc[team] = [accuracy]

Here I used a for loop to create a seperate training and test set for each team and ran the algorithm for each team. I then extracted the accuracy of each run and stored it in a seperate dataframe. Now I will graph the results and I'll discuss my findings.
